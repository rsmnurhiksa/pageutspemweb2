ask Manager API (UTS Pemrograman Web 2)
📌 Deskripsi

Backend sederhana untuk aplikasi Task Manager menggunakan Node.js, Express, dan PostgreSQL.

⚙️ Teknologi
Node.js
Express.js
PostgreSQL
pg
Nodemon
🚀 Cara Menjalankan

git clone https://github.com/username/task-manager-api.git
cd task-manager-api
npm install
npm start

📮 Endpoint API
🔹 GET /tasks

Mengambil semua task




🔹 GET /tasks/

Mengambil task berdasarkan ID




🔹 POST /tasks

Menambahkan task baru




🔹 PUT /tasks/

Update task




🔹 DELETE /tasks/

Menghapus task
